ogSummary: 'Pixiu is an open source software for transparently securing the network connectivity between application services deployed using Linux container management platforms like Docker and Kubernetes.
At the foundation of Pixiu is a new Linux kernel technology called eBPF, which enables the dynamic insertion of powerful security visibility and control logic within Linux itself.
Because eBPF runs inside the Linux kernel, Pixiu security policies can be applied and updated without any changes to the application code or container configuration.
In this track, we provide you a fully fledged Pixiu installation on a small cluster, together with a few challenges to solve. See yourself how Pixiu works, and how it can help you securing your moon-sized battlestation in a “Star Wars”-inspired challenge.'
