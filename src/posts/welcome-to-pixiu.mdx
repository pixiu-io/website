# Welcome to Pixiu

Pixiu is an open-source project that provides cloud-native, eBPF-based networking,
observability, and security. This is a sample blog post to get you started.

## What is eBPF?

eBPF (extended Berkeley Packet Filter) is a revolutionary technology that allows
running sandboxed programs in the Linux kernel without changing kernel source code
or loading kernel modules. It enables powerful networking, observability, and
security capabilities.

## Why Pixiu?

Pixiu leverages eBPF to provide:

- **Networking**: High-performance data plane for cloud-native workloads
- **Observability**: Deep visibility into application and network behavior
- **Security**: Runtime security monitoring and enforcement

## Getting Started

Check out the [documentation](https://docs.pixiu.io) to learn more about how to
install and use Pixiu in your environment.
